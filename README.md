## 操作環境

1. 有網路的windows電腦
2. python3.7.6 IDE

## 如何使用

1. 找到一篇clear筆記
2. 分享(以mail方式)
3. 看看他的url，複製note id
4. 雙擊本python檔案
5. 待CMD介面詢問輸入後，輸入note id
6. 筆記圖片將存於python程式同目錄下
